## Installation

Clone the project

```bash
  git clone https://github.com/kevinbalboni/openui5-odata-visualizer.git
```
Go to the project directory

```bash
  cd openui5-odata-visualizer
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
